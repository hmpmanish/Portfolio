# Manish Pandey Portfolio

Welcome to my personal portfolio! This project showcases my skills, experience, and the projects I have worked on. It is designed to highlight my abilities as a web developer and serves as a platform for visitors to connect with me.

---

## 🚀 Features

- **Responsive Design**: The portfolio is mobile-friendly and adapts to various screen sizes.
- **Dynamic Animations**: Includes engaging animations to enhance user experience.
- **About Me Section**: Provides an introduction to my background and expertise.
- **Social Media Integration**: Links to my LinkedIn, GitHub, Instagram, and Twitter profiles.
- **Contact Form**: A functional form for visitors to send messages directly to me, integrated with SheetDB.

---

## 🔧 Technologies Used

- **HTML5**: Structure of the portfolio.
- **CSS3**: Styling and animations.
- **JavaScript**: Interactive functionality, including the contact form integration.
- **SheetDB**: API for managing form submissions.

---

## 📁 Project Structure

```
.
├── index.html         # Main HTML file for the portfolio
├── README.md          # Documentation file
├── assets/            # Folder for images, icons, and other assets
├── styles/            # Folder for additional CSS files (if any)
└── scripts/           # Folder for JavaScript files (if needed)
```

---

## 🌟 How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/hmpmanish/portfolio.git
   ```
2. Navigate to the project folder:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your browser to view the portfolio.
